Chapitre 2 : Polynômes du 2nd degré

Définition :

  • On appel fonction polynôme du second, toute fonction : f(x) = ax2+bx+c où (a,b,c)∈ℝ* ✖ ℝ2
  • a est le coefficient du terme de plus haut degré.
    b est le coefficient du terme de degré 1.
    c est le coefficient constant.

I) Forme canonique :

  • f : x ↦ ax2+bx+c, avec a ≠ 0 peut s'écrire sous forme canonique :
    f(x) = a(x-α)2+β avec α = -b2a et β = f(α) = f(-b2a)

II) Discriminant :

  • On appel discriminant de la fonction f : x ↦ ax2+bx+c avec a ≠ 0 le réel Δ :
    Δ = b2-4ac
  • la forme canonique s'écrit alors:
    f(x) = α(x+-b2a)2-Δ4a

III)Racines et factorisation éventuelle :

Définition :

  • On appel racine de la fonction polynôme f du second degré toute solution de l'équation f(x)=0

Théorème :

Soit f : x ↦ ax2+bx+c avec a ≠ 0 alors :
  • Si Δ > 0, f admet 2 racines, x1 et x2 avec x1 ≠ x2 :
    x1 =-b-sqrt(Δ)2a et x2 =-b+sqrt(Δ)2a
    Et pour tout réel x, f(x) = a(x-x1)(x-x2)

  • Si Δ = 0, f admet une unique racine(dite double), x0:
    x0 =-b2a
    Et pour tout réel x, f(x) = a(x-x0)2

  • Si Δ < 0, f n'admet pas de racine réelle et ne peut pas être factorisé dans ℝ

IV) Signe de f(x) :

Soit f : x ↦ ax2+bx+c avec a ≠ 0 alors :
  • Si Δ > 0, notons x1 et x2 avec x1 < x2 les racines de f.
