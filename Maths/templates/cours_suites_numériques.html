Chapitre 1 : Suites numériques

Définition :

  • Une suite est une fonction U : I ➝ ℝ avec I⊆ℝ
  • L'image de n par la fonction U est noté Un ou U(n)
  • On dit que Un est le terme d'indice n de la suite

  • Si le premier terme de la suite est U0,alors Un est le (n+1)-ième terme de la suite
  • Si le premier terme de la suite est U1,alors Un est le n-ième terme de la suite

Définir une suite :

Une suite est généralement définie :
  • Soit par une relation du type : Un=f(n)
  • Soit par son premier terme (Up) et une relation de récurence : Un+1=f(Un)

Sens de variation d'une suite :

  • Une suite est strictement croissante si ∀n, Un < Un+1
  • Une suite est strictement décroissante si ∀n, Un > Un+1

I)Suite arithmétique :

  • Une suite arithmétique est une suite définie par : Up et Un+1 = Un+r où r est appelé raison de la suite
    ⇔Une suite est arithmétique si est seulement si Un+1-Un = r, r étant une constante.
  • Soit (Un), une suite arithmétique de raison r et de premier terme p : Un = Up+(n-p)r
  • Les formules donnant Un en fonction de n sont celles de fonctions affines de coefficient directeur r
  • Soit (Un) une sutie arithmétique de raison r :
    -r > 0 ⇔ (Un) est strictement croissante
    -r < 0 ⇔ (Un) est strictement décroissante
    -Tout les points de coordonées (n;Un) sont alignés
  • Soit (Un) une suite arithmétique de raison r. Notons Snla somme de tout les membres d'indice inférieur ou égal à n alors :
    Sn= (n-p+1)(Up + Un)2

II)Suite géométrique :

  • Une suite géométrique est une suite définie par : Vp et Vn+1 = Vn ✖ r où r est appelé raison de la suite
    ⇔Une suite dont tout les termes sont non-nuls est géométrique si est seulement si Vn+1Vn = r, r étant une constante.
  • Soit Vn, une suite géométrique de raison r et de premier terme p : Vn = Vp ✖ rn-p
  • La suite géométrique (Vn) de raison r est strictement croissante ⇔ r > 1
    La suite géométrique (Vn) de raison r est strictement décroissante ⇔ r ∈ ]0;1[
  • Une suite géométrique de raison r > 0 a une croissante ou décroissante est dites exponentielle
  • Soit (Vn) une suite géométrique de raison r ≠ 1 et de premier terme p. Notons Sn la somme de tout les termes d'indice ≤ n alors :
    Sn = 1-q^(n+1)1-q
